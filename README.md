# Door2Door

Door2Door helps the users to find the vendors near by their home location whenever that vendor is within area  of about 1-2 km radius. 

Currently this app is targeted for users in India and in India small vendors are mobile and move places based on their sale. 

This app vendors are able to alert the users of that location of their availability so users can look for them and purchase the goods at low cost by helping themselves and the vendors thereby encouraging small businesses. 

This app is still in the basic development and is hosted on Github, 

Techstack - Expo and React Native are the prmiary development frameworks . For database , it is preferred to use Mongo DB or other No SQL database. This app will be hosted on AWS.
